# Flutter Mobile App (Controller)

This Flutter app can:
- Add task (title + time)
- View tasks
- Send POST request to backend
- Fetch tasks with GET

## Setup

1. Go to this folder.
2. Run:
   flutter create .
3. Run:
   flutter pub get
4. Open lib/main.dart and update baseUrl:
   http://YOUR_SERVER_IP:5000/api
5. Start app:
   flutter run

## Time Format

Use HH:MM format, for example 09:15 or 18:45.
